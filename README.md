# crud
CRUD  de usuarios com nodejs e Postgres

1 - Instale o node.js em seu computador;
2 - Instale o Postgres em seu computador, configure logine senha e nome do banco.
3 - Clone o projeto para seu repositório local;
4 - No diretório do projeto, abra o CMD;
4 - Rodar a migration: yarn sequelize db:migrate;
5 - Iniciar a API com o comando: yarn dev;
6 - Instale o Insomnia em seu computador;
7 - Importar o arquivo crudUsers localizado dentro da pasta doc, para fazer as requisições na API.
